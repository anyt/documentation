Add All Products Page to Frontend Menus per Website
---------------------------------------------------

.. hint:: This topic is part of :ref:`Concept Guides: All Products <sys--conf--commerce--catalog--special-pages>`, where you can check out a sample of adding All Products page to the storefront and examine other All Products configuration options.

To add the All Products page to the frontend menu of a particular website:

1. Navigate **System > Websites** in the main menu.
2. Click once on the required website to open its page.
3. On the website page, click |IcConfig| **Edit Frontend Menu** to start editing the configuration.

   .. image:: /user_doc/img/products/all_products_page/AllProductsWebsiteEditMenu.png

3. Click once on the menu to which you will add the All Products page.
4. Click **Create Menu Item** on the top right of the page.
#. In the **Title** field, type in the label for the menu item.
#. In the **URI** field, specify */catalog/allproducts*.
#. Complete the other fields as required.

   .. image:: /user_doc/img/products/all_products_page/AllProductsWebsite.png

#. Click **Save** on the top right of the page.

The All Products page should now become available as part of the selected menu on the website level.
