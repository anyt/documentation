MS Exchange Integration Settings per Organization
=================================================

To configure MS Exchange integration settings in your Oro application:

1. Navigate to **System > User Management > Organizations** in the main menu.
2. For the necessary organization, hover over the |IcMore| **More Options** menu at the end of the row and click |IcConfig| to start editing the configuration.
3. Click **System Configuration > Integrations > MS Exchange Settings** in the menu to the left.
4. To enable MS Exchange Integration for the current organization, select the **Enabled** check box and define the following details:

   * **Server** --- Enter the name of your Microsoft Exchange Server instance.
   * **Version** --- Choose the server version from the drop-down menu.
   * **Login and Password** -- Enter the credentials of the super-user.
   * **Domain List** --- Define the domains to which you will grant access. At least one domain **must** be defined.

5. Click **Save Settings**.
