Configure Google Hangouts Integration per Organization
======================================================

.. hint:: You can read more on this topic in :ref:`Voice and Video Calls via Hangouts <user-guide-hangouts>`.

To enable hangouts in the Oro application for an organiation:

1. Navigate to **System > User Management > Organizations** in the main menu.
2. For the necessary organization, hover over the |IcMore| **More Options** menu at the end of the row and click |IcConfig| to start editing the configuration.
3. Click **System Configuration > Integrations > Google Settings** in the menu to the left.
4. In the **Google Hangouts** section, select the following check boxes:

   * **Enable for Emails** --- to call via email (you can invite up to 5 users to a Google Hangout by emailing them)
   * **Enable for Phones** --- to make phone calls

5. Click **Save Settings**.
