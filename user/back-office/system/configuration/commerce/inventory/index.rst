Inventory
=========

Discover how to configure your storefront and back-office inventory statuses, add warehouses, manage product inventory options, set inventory thresholds and quantity limitations. You can control the way inventory is managed in OroCommerce on three levels -- globally, :ref:`per organization <configuration--commerce--inventory-organization>` and :ref:`per website <configuration--commerce--inventory-website>`.
